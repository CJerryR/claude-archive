# 安全策略

## 隐私与数据

Claude Archive Suite **完全在本地运行**：

- 所有捕获的对话与文件仅写入你本机浏览器的下载目录（`ClaudeArchive/`）。
- 扩展**不向任何第三方服务器发送数据**。
- 唯一的外部网络请求来自**本地查看器**通过 CDN 加载 KaTeX（用于公式渲染）；如需完全离线，可改为本地引入 KaTeX。
- 扩展仅在 `claude.ai` / `*.claude.ai` / `claude.hk.cn` 域名下激活，仅访问归档所需的对话与文件接口。

## 权限说明

| 权限                           | 用途                                                           |
| ------------------------------ | -------------------------------------------------------------- |
| `storage` / `unlimitedStorage` | 保存设置、对话索引、已存文件记录、运行日志                     |
| `downloads`                    | 将对话与文件写入下载目录；查询文件是否已存在以避免重复下载     |
| `tabs`                         | 定位 Claude 标签页以代理带凭证的抓取；批量保存时导航到指定对话 |
| `offscreen`                    | 在 Service Worker 中把文本/二进制转换为可下载的数据 URL        |
| `host_permissions`             | 仅限 Claude 官方域名与已声明的镜像站                           |

## 支持的版本

仅最新版本接受安全修复。请保持更新到最新发布版。

| 版本  | 是否支持 |
| ----- | -------- |
| 2.7.x | ✅        |
| < 2.7 | ❌ 请升级 |

## 报告漏洞

如发现安全问题，请**不要**公开提交 Issue，改为通过 GitHub 的私密漏洞上报（Security Advisories）或私信维护者。请附：

- 受影响的版本与浏览器；
- 漏洞描述与复现步骤；
- 可能的影响范围。

我会尽快确认并修复。
