Claude 存档插件 · 快速上手

把你和 Claude 的聊天（含思考过程、文件）自动存到电脑，用仿官网界面离线翻看
🔒 全程在你电脑本地运行，不上传任何聊天内容
第一部分：安装（约 5 分钟，只做一次）
1

解压文件

右键收到的压缩包 → 「全部解压缩」。

得到一个文件夹，里面有 manifest.json 就对了。

⚠️ 记住位置，以后别删别移动。

2

打开扩展页

用 Chrome：地址栏输入 chrome://extensions 回车。

用 Edge：地址栏输入 edge://extensions 回车。

3

开「开发者模式」

Chrome 在页面右上角、Edge 在左下角，

找到「开发者模式」开关，点开（变蓝）。

4

加载插件

点上方出现的 「加载已解压的扩展程序」，

选中第 1 步那个文件夹，确定。

出现插件卡片 = 成功 ✅

5

固定图标到工具栏

点浏览器右上角的拼图图标 🧩 → 找到「Claude Archive」→ 点它旁边的图钉 📌。以后点这个小图标就能打开插件面板。

💡 装完就行了，平时什么都不用做。正常聊天，插件会在后台自动保存，存好时右上角图标闪一下 ✓。

📌 第二部分：日常使用

  1. 自动保存：照常用 Claude 聊天即可，记录自动存进电脑「下载 → ClaudeArchive」文件夹。
  2. 旧对话没存上？ 打开那个对话，按一下 F5 刷新网页，就能抓到了。
  3. 翻看记录：点插件面板里的 「打开查看器」 → 点「选择存档文件夹」→ 选「下载」里的 ClaudeArchive 文件夹。左边列出所有对话，点开即可看。
  4. 想一次性存全部：插件面板点 「全部下载」；想检查有没有漏存的文件，点 「检查保存完整性」。
❓ 常见疑问：
· 找不到图标 → 点右上角拼图 🧩 里面有。
· 浏览器问"要停用开发者模式扩展吗" → 点取消/保留，别停用。
· 插件突然没了 → 多半是第 1 步的文件夹被删/移动了，放回原处或重装。
· 图片是 .webp → 正常，浏览器能直接打开。
全部数据只存在你自己电脑上 · 遇到问题点「导出日志」把文件发给分享插件给你的人即可排查
