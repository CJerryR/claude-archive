# Release v2.0.0

**发布日期**: 2026-06-11

### ✨ 新增（Added）
- 新增数学公式渲染：支持 `$$...$$` 与 `$...$`，经 KaTeX 渲染（CDN 加载失败时回退为可读原文）。
- 代码块新增「一键复制」按钮与语言标签。
- 新增 VSCode Dark+ 风格代码语法高亮（自写分词器，无外部依赖）。
- 每条消息底部新增页脚：显示最后更新时间（精确到秒）与该条 Claude 思考用时。
- 新增「统计」页面：对话数、消息与字数、累计思考时长、工具调用排行、最忙日期、跨越时间等。
- 扩展弹窗新增每对话文件下载进度条。

### 🔧 改进（Changed）
- 引用块样式改为 Claude 官网风格（浅色卡片 + 赤陶色左边条）。
