# Release v2.2.0

**发布日期**: 2026-06-11

### ✨ 新增（Added）
- 新增「每轮产出独立文件夹」机制：Claude 各轮产出按 `files/r{序号}_{时分秒}_{消息uuid8}/` 归档（**注:该模型已于 2.5.0 被内容指纹版本化取代**）。
- 查看器新增按轮次解析产出文件：每条消息的产出优先读取其所属轮次目录中的版本。

### 🔧 改进（Changed）
- 产出文件名改为按响应真实 `Content-Type` 推断扩展名（预览图为 webp 即存 `.webp`），不再强制改名。
- 生成文件名优先取 `file_path` 的真实文件名（含扩展名），修正此前丢失扩展名的问题。

### ⚡ 优化（Improved）
- 跟踪记录、下载进度、统计数据持久化于 `chrome.storage.local`，更新扩展版本后仍保留。
