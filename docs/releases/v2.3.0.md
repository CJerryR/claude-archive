# Release v2.3.0

**发布日期**: 2026-06-11

### ✨ 新增（Added）
- 新增独立工具 `整理报告.html`：扫描本地 `ClaudeArchive`，按对话 + 版本归类，标注每个文件的布局来源、所在子目录与大小；以 SHA-256 识别内容重复文件；同名多版本标注「N 版」；可导出 JSON / CSV 清单。**全程只读，不修改任何文件。**

### 🔧 改进（Changed）
- 查看器向下兼容全部历史目录布局（平铺 `files/`、对话码 `files/{码}/`、轮次 `files/r.../`）：按文件名在该对话 `files/` 子树内查找，找不到再回退，旧存档无需整理即可正常打开。
