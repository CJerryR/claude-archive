# Release v2.4.0

**发布日期**: 2026-06-11

### ✨ 新增（Added）
- 新增 `整理手册_给ClaudeCode.md`：面向 Claude Code 的历史文件整理规则文档，涵盖全部历史命名/布局规律、去重规则、目标结构与可执行步骤。

- 修复反复保存同一对话时产生 `__2`/`__3` 等冗余副本的问题。
