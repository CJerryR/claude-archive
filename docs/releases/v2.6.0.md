# Release v2.6.0

**发布日期**: 2026-06-11

### ✨ 新增（Added）
- 新增「全部下载（所有已跟踪对话）」功能：遍历本地已跟踪的全部对话，在后台逐个加载并保存记录与文件，带实时进度条。
- 新增「检查保存完整性」功能：逐个核对每个对话应有文件与本地已存文件，自动补下缺失文件；全部齐全时提示「无缺失」。
- 新增后台能力 `navigateAndCapture`：对不在内存中的历史对话，自动将 Claude 标签页导航至该对话并等待捕获，使其也能被归档。
- 为查看器与扩展弹窗的每个按钮新增鼠标悬停说明（`title`）与文字简介。

### 🔧 改进（Changed）
- 思考链折叠标题改为智能回退：优先使用思考摘要（`summaries`），无摘要时回退使用该步骤首个工具的动作描述（`message`），仅在均无时才显示「思考」。
- 弹窗「已跟踪」数量改为从持久存储（`index`）读取。

### 🐛 修复（Fixed）
- 修复「已跟踪」列表在刷新或扩展重启后被清零的问题（此前读取的是内存态 `state.size`）。
