# Release v2.7.0

**发布日期**: 2026-06-12

### ✨ 新增（Added）
- 新增「调试日志」开关:开启后在 Service Worker 控制台打印「追踪 → 保存」每一步,便于排查对话追踪不上的问题。
- 新增运行日志环形缓冲(保留最近 300 条)与「导出日志」按钮,导出至 `ClaudeArchive/_runlog.txt`。
- 在捕获、归档开始/跳过/完成、补全抓取等关键节点埋设日志点。

### ⚡ 优化（Improved）
- 日志会明确说明跳过原因(未捕获到数据、内容未变化、开关关闭等),便于自助定位。
