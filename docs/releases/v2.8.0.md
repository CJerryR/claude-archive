# Release v2.8.0

**发布日期**: 2026-06-12

### 🔧 改进（Changed）
- **[破坏性变更]** 重写产出文件的保存与版本规则,严格遵循:① 绝不覆盖任何已存文件;② 内容不同需保存时,先查已占用文件名,撞名则递增 `__v2`/`__v3`;③ 内容相同(SHA-256 一致)直接跳过不存。版本号改为基于扩展自身的持久记录判定,不再依赖会被清除的浏览器下载历史。

### ✨ 新增（Added）
- 新增「静默下载」开关(默认开):文件下载完成后从浏览器下载记录中移除(文件保留在磁盘),避免下载栏被刷屏。
- 查看器搜索对话时,标题中命中的关键词以高亮样式(赤陶色文字+浅底+圆角边框)标出。

### 🐛 修复（Fixed）
- 修复反复保存时仍弹出 Windows「文件已存在,要替换吗」系统窗口的问题(改用基于自身记录的命名 + `uniquify` 兜底,永不弹窗)。
- 修复版本号混乱(出现 `__v3` 反而比 `__v2` 旧/大等)的问题。
- 修复查看器中点击对话、切换分支时视图跳到顶部的问题:打开对话定位到最新消息,切换分支定位到该分支对应消息。
- 行内代码样式由近黑高亮改为赤陶色文字+浅底+圆角边框,深浅主题一致。
